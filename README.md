DIServer
========
一个由swoole_server提供网络服务，ThinkPHP提供ORM等其它工具的TCP/UDP网络服务框架。
-----------------------------------------------------------------
特点：

1、可以与ThinkPHP原有的Web功能融合。

2、对ThinkPHP的部分代码做了一定的修改（因为这个版本的ThinkPHP是我用了很久而且一直在改的所以暂时没整理出改了什么）。
